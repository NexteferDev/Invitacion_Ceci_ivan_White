# 💍 Invitación Digital: Ivan & Cecy

Una aplicación web moderna, elegante y personalizada para nuestra boda. Construida con **Next.js 14**, **Tailwind CSS**, **Framer Motion** y **Firebase** para la gestión de confirmaciones en tiempo real.

![Vista Previa](public/thumbnail.png)

## ✨ Características Principales

### 🎨 Diseño y Experiencia
- **Estética Elegante**: Paleta de colores en Negro Elegante (`#1a1a1a`), Oro (`#d4af37`) y Crema (`#faf8f3`).
- **Animaciones Premium**: Transiciones suaves y elementos flotantes con Framer Motion.
- **Totalmente Responsivo**: Optimizado para que los invitados lo vean perfecto en sus celulares.

### 🛠️ Funcionalidades Técnicas
- **RSVP con Firebase**: Formulario de confirmación conectado a Firestore Database para recibir respuestas al instante.
- **Cuenta Regresiva**: Contador dinámico para el 02 de mayo de 2026.
- **Música de Fondo**: Reproducción automática optimizada para navegadores modernos.
- **Menú Flotante**: Navegación inteligente que aparece al hacer scroll.
- **Google Maps**: Integración de ubicación para la ceremonia y recepción.

## 🚀 Tecnologías Utilizadas

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Base de Datos**: [Firebase Firestore](https://firebase.google.com/)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Iconos**: [Lucide React](https://lucide.dev/)
- **Despliegue**: [Vercel](https://vercel.com/)
